.NET JWT Authentication API

A simple and clean authentication API built with ASP.NET Core, implementing user registration, login, and JWT-based authentication.

This project focuses on core backend fundamentals such as secure authentication flow, token handling, and API structure.

---
Features
- User registration (Sign Up)
- User authentication (Login)
- JSON Web Token (JWT) generation
- Protected endpoints using JWT authorization
- Basic password handling (hashing if implemented)
- Clean and modular API structure

---
Notes

This project is part of a larger backend learning journey and focuses specifically on authentication and authorization concepts.
